JDK:(JRE+JVM)
Java development kit where we can write our Code to develop a application using Development tool that help us to compiling, document and debugging
Here Javac converts the java code into Byte code. 

JRE:(contains JVM)
Java Runtime Environment helps to run the program and contains Core Java Libraries 

JVM:
Which make java as platform independent by executing byte code in any platform such as Linus or Windows os.

**Write Once, Run Anywhere** means Java code can be written one time and executed any time in any OS without modification if that system as JVM
**Byte Code** is the intermediate and  platform independent code generated by Java compiler. Byte code is not human-readable. Executing in byte code help improve performance as well where JIT converts Byte code to Machine Code.
